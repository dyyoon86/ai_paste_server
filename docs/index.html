내 PC 확인 중…

이렇게 쓰면 돼요

  1. 스킬 파일을 claude.ai에 올리고, 만들고 싶은 영상 주제를 한 줄 입력해요.
    📄 스킬 파일 받기
  2. claude가 만들어준 JSON을 복사해요.
  3. 이 앱을 열고 붙여넣기 → 만들기. 끝! MP4가 나와요.
처음 실행할 때 윈도우/맥이 "확인되지 않은 앱"이라고 한 번 물어봐요(서명 비용 때문). 윈도우는 추가 정보 → 실행, 맥은 우클릭 → 열기 한 번만 누르면 돼요. 그다음부턴 안 물어봐요.
