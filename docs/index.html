How it works
1

프롬프트를 AI에 붙여넣기

ChatGPT·Gemini·Claude 어디든 프롬프트를 붙여넣고, 만들 영상 주제를 한 줄 적어요.

프롬프트 보기
또는 Claude Skills 쓰면 → 스킬 .zip 받기
2

나온 JSON 복사

claude가 만들어준 영상 설계(JSON)를 복사해요.

3

앱에 붙여넣기 → 만들기

이 앱을 열고 붙여넣은 뒤 버튼 한 번. MP4가 나와요.

⚠️
처음 실행 때 윈도우/맥이 "확인되지 않은 앱"이라고 한 번 물어봐요(서명 비용 때문). 윈도우는 추가 정보 → 실행, 맥은 우클릭 → 열기 한 번이면 끝.
